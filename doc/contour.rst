The Contour data structure
==========================

The basic data structure in MusiContour is the class :class:`Contour`.

Contour methods
---------------

.. class:: Contour()

.. method:: inversion()

The method :method:`inversion` is based on Jamary Oliveira function:

  I(x)n = 2n - x

Robert Morris uses this function:

  I(x)n = (n - 1) - x
